## Project Description
This project is part of a Flutter practical assignment focused on understanding the fundamentals of building mobile user interfaces.

In this project, several key concepts are implemented, including:
- Basic widgets such as Text, Container, Image, Icon, and Button
- Layout management using Column, Row, Stack, Expanded, and Spacer
- Use of Card and ListTile to display structured information
- Implementation of a simple profile page (ProfilPage)
- Understanding the difference between StatefulWidget and StatelessWidget
- Applying custom themes (light mode and dark mode)

Additionally, this project emphasizes writing clean and organized code by separating different pages into multiple files, such as:
- main.dart
- profil_card.dart
- layout_demo.dart
- profil.dart
  
The main objective of this project is to build a solid foundation in Flutter development and create responsive and visually appealing user interfaces.
